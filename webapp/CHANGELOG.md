# 更新日志 · Changelog

本项目（个股看板 Stock Dashboard）遵循[语义化版本](https://semver.org/lang/zh-CN/)。

格式：`主版本.次版本.修订号`
- **主版本**：不兼容的重大改动（数据结构/部署方式变化等）
- **次版本**：向后兼容的新功能
- **修订号**：向后兼容的问题修复

Git 标签以 `webapp-v<版本>` 命名（避免与仓库内 skills 的 `v*` 发布标签冲突）。

## [1.3.0] - 2026-07-01

### 新增:枢轴跟踪池 + 买点识别(plan #3)
- 新增**「📍 枢轴跟踪」页**:TradingView 选出的候选粘进来,系统每天盘后跟踪是否接近/突破枢轴并提醒
- **手填枢轴 + 盘后跟踪**:算距枢轴% + 量能比,判定 5 类信号——👀接近(枢轴下方3%内)/ 🚀放量突破(量≥1.5×50日均量,可买)/ ⚠️缩量突破(存疑)/ 🈵已过可买区(追高)/ ❌跌破参考止损(出局)
- **提醒**:盘后 Telegram 推送 + 站内 banner,`notify_state` 去重、状态回落自动重新武装
- **VCP-lite 自动建议枢轴 + 基座质量**:分形识别整理基座,给出建议枢轴 + 基座长度/回调深度/收缩序列(如 25→15→8)/是否收紧/缩量/Stage2;「用建议」一键采纳并确认
- **K 线叠加**:个股看板叠加枢轴线 + 可买上限 + 参考止损;跟踪池支持 TradingView 批量粘贴导入、行内改枢轴、已买/移出

### 数据模型
- 新增 `tracker` 表(含幂等迁移);跟踪池标的纳入盘后 K 线增量刷新

### 技术
- `_evaluate_tracker` / `_base_stats`(VCP-lite)/ `_tracker_check_and_push`;`GET/POST /api/tracker`、`PUT/DELETE /api/tracker/<id>`、`GET /api/tracker/pivot`

## [1.0.0] - 2026-07-01

首个正式版本。Flask + yfinance 的个人炒股决策仪表盘，四大页面 SPA，数据落盘 SQLite，经 Cloudflare Tunnel + Access 部署在 ECS。

### 个股看板
- 一键决策卡（SEPA + 估值 + 期权墙 → 买入/观察/回避 + 建议仓位）
- K 线（4h / 1d / 1w）+ MA5/10/20/50/200 + 成交量 + 可选期权墙位叠加
- SEPA 趋势模板评分、财报日/业绩、流动性评分、关键财务、分析师评级、新闻流
- 价格 / 跌破 20MA / 触及止损 实时预警（后端评估，触发顶部 banner，可经 Telegram 推送）
- 估值三角定位（DCF + 分析师目标 + 远期 PE）、分析师预期与 EPS 修正
- 期权墙：Max Pain、Call/Put OI 支撑压力墙、净 GEX + Gamma Flip、Put/Call 比率
- 多股对比：归一化走势、区间涨幅/波动/Beta、相关性矩阵
- 个股研究笔记（按标的归档）

### 仓位计算器
- 按买入价/止损/总资产 + 仓位上限 + 风险上限算最大可买股数
- **可手改买入股数**（默认最大值），派生金额实时重算；改任一条件即重置回最大值
- 一键「记入持仓」，按实际填入的股数记录

### 持仓 / 跟踪
- 总资金量持久化，作为唯一真相；显示已投入/可用现金/浮盈亏/**仓位占账户（按市值）**/组合风险敞口
- 持仓表：实时浮盈亏、**距止损（回落跌幅，绿赚红亏）**、**风险敞口（$）**、备注
- 行内编辑（股数/成本/现价/止损/目标/备注）；编辑单笔买入的仓位会同步修正其交易明细
- **加仓自动合并**到同标的仓位（加权平均成本）
- **手填现价**支持无实时行情的标的（期权 / 港股等）
- **空头支持**（负股数，如卖出开仓/备兑期权）：盈亏方向、平仓（全平/部分）均正确
- **平仓**：行内表单输入平仓价 + 数量，支持部分平仓（保留剩余仓位并注明）与清仓（记录累计已实现盈亏）

### 交易明细（买卖流水台账）
- 每笔买入/卖出独立记录，带精确时间戳，**严格按成交先后倒序**（最新在最上面）
- 分页显示，每页 20 条；可单条删除
- 已实现盈亏逐笔记录，清仓时汇总该标的总盈利

### 每日复盘
- **「今天」实时显示**当前组合（改总资金量/价格/交易即时反映）
- 每个交易日 **17:00（美东）自动冻结**当日存档（持仓、交易、盈亏、敞口），历史日期为只读快照
- 每日可写复盘文字并持久化，供日后回查
- 「立即冻结/更新今日存档」手动按钮

### 市场热力图
- 个股热力图（按板块分组 treemap，面积≈市值，色=当日涨跌）
- 板块热力图（市值加权）；板块可自定义编辑

### 部署 / 基础设施
- Docker 化，仅绑回环 `127.0.0.1:8799`；Cloudflare Tunnel 出站 + Access 零信任登录
- SQLite（WAL）持久化，K 线缓存跨重启；收盘后后台刷新重点标的
- 盘后总结 / 告警经 tv-relay 转 Telegram
