## 三、发现的问题(按重要性)

### 🔴 1. 组合风险敞口低估真实风险(安全隐患)—— 已修复提示
后端 `total_risk` 只累加**设了止损**的持仓;未设止损的持仓按 $0 计入。你当前 8 个持仓里 **5 个没有止损**(含 3 个卖出开仓的期权,下行/上行风险很大),所以「组合风险敞口 3.69%」把风险**最不封顶**的那几笔恰好排除在外,数字会让你误以为很安全。
- **已做的修复**:风险敞口卡片现在会显示 `⚠ N 个无止损` 角标并在 hover 说明「实际组合风险更高」。
- **建议进一步**:对 Minervini 策略,「每一笔都必须有止损」是铁律 —— 可考虑给无止损持仓一个更醒目的红色 banner,或在记入持仓时强制要求填止损。

### 🔴 2. 仓位计算不拦「追高」入场 —— 已修复提示
默认止损取「最靠近且低于现价的均线」。像 MXL 当天大涨、股价远在 20MA 之上时,默认止损给到 **-25%**,而工具只在小字里写了「7-8%」,不会阻止你按 25% 止损去建仓。右侧交易的精髓正是在**枢轴附近**入场,让止损天然收窄到 7-8%。
- **已做的修复**:止损距离 >10% 时,计算器顶部弹出橙色警告,提示这是追高、应等回踩或换更贴近支撑的入场点。

### 🟠 3. 相对强度(RS)条件偏宽松(准确性)
SEPA 第 8 条目前的判定是「个股 252 日涨幅 > 标普500」。但 Minervini 的原版要求 **RS Rating ≥ 70**(相对**全市场**的百分位排名)。一只股票可以跑赢标普、但 RS 百分位只有 50–60。结果:趋势模板容易报「8/8 · 强势候选」,把 Minervini 其实会淘汰的票也放进来。
- 代码里已诚实标注这是「代理」。若想更贴近原版,可用自选池/一个较宽的股票宇宙算百分位,或至少要求「跑赢标普 + RS 线创新高」。

### 🟡 4. 缺少「买点 / VCP / 基座突破」识别(最大的策略缺口)
这是最重要的一条。工具能告诉你「这只票在 Stage 2 上升趋势」,但**没告诉你在哪买**——没有 VCP(波动收缩)、没有枢轴/基座突破、没有「缩量整理 + 放量突破」的量价确认。右侧交易的下单动作,几乎完全取决于「基座末端的枢轴买点」,而这一块目前是空白(Roadmap A 项部分涵盖)。

### 🟡 5. 缺少交易日志与绩效统计(直接关系到「提升」)
你的目标是**提升**。提升来自复盘决策本身:入场理由、setup 类型、情绪、对错、亏损单的事后剖析。现在的笔记是「按票」「按天」的,还没有把 **一笔交易的 setup → 结果 → 教训** 串起来的结构化日志,也没有 Minervini 极度强调的绩效指标:**胜率、平均盈利/平均亏损、盈亏比、期望值(expectancy)、批打率**。(Roadmap B 项涵盖日志,但统计还没有。)

### ⚪ 6. 资金曲线起点为空(历史丢失)
快照从 V1.0.1 才开始,之前的账户历史无法自动重建。影响不大,可手动补录几个关键节点。

## 四、这个工具「够不够」达成你的目标?

**记录层面:够。** 持仓、流水、每日复盘、资金曲线四件套齐全且做得扎实,足以完整留痕。

**提升层面 + 贴合 Minervini:部分够,还差「闭环」。**
它把「趋势模板筛选」这一步做得不错,但右侧交易真正决定成败的两件事——**精确买点(VCP/枢轴)** 和 **止损纪律的强制执行**——目前一个缺失、一个偏软(本次已加提示)。而「提升」需要的反馈闭环(setup 打标 → 绩效统计 → 事后复盘)也还没成形。

一句话:**它现在是一个很好的「看板 + 记录器」,但还不是一个「让你变强的教练」。** 补上下面第五节的三件事,就能真正对上你的目标。

---

## 五、建议(按性价比排序)

1. **止损纪律硬化**(小改,最高价值):无止损持仓给红色 banner;记入持仓时止损必填;组合热度旁常驻提示。方向已由本次两处修复开头。
2. **交易日志 + 绩效面板**(中等工作量,直接服务「提升」):新表 `journal`,每笔交易记 setup 类型/入场理由/情绪分/退出理由;自动算胜率、盈亏比、期望值、平均持有天数,并按 setup 类型分组看哪类 setup 最赚钱。这是 Minervini 式复盘的核心。
3. **买点识别 / VCP 标注**(较大,策略核心):在 K 线上自动标注基座、收缩、枢轴,并给「距枢轴 %」「突破是否放量」。哪怕先做**单标的的视觉标注**(不做全自动),对右侧择时也极有帮助。Roadmap A 项可优先做这一小步。
4. **RS 收紧**:改为相对自选池/宽宇宙的百分位,或加「RS 线新高」确认,让 8/8 更可信。
5. **资金曲线补录**:手动补几笔历史总资金,让曲线从有意义的起点开始。
